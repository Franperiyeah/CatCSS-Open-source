Getting Started

CatCSS is a lightweight and modular CSS framework. Follow these steps to get started:

  1. Clone the repository or download the latest release.
  2. Include the dist/catcss.css file in your project:
    3. <link rel="stylesheet" href="dist/catcss.css">

Customization

CatCSS is fully customizable using CSS variables. You can override these variables in your own project:

:root {
  --primary-color: #ff0000;
  --font-size-base: 18px;
}
